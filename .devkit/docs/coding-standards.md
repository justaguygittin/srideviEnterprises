# Sridevi Enterprises Coding Standards

## Python
- Use snake_case
- Write docstrings for functions
- Add comments only when they explain intent

## HTML
- Use kebab-case for IDs and classes
- Group sections with comment dividers
- Use template inheritance

## CSS
- Use CSS variables
- One responsibility per file
- Keep selectors simple

## Git
- feat:
- fix:
- docs:
- style:
